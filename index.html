💡 使用说明

1. 本系统为原型演示，三端数据通过浏览器 localStorage 共享，实现联动效果。
2. 建议同时打开三个页面（用户端、商家后台、统计后台），体验数据实时同步。
3. 用户端模拟微信小程序界面，商家后台和统计后台为 Web 管理系统风格。
4. 异常场景演示：库存不足、支付失败（用户端有"模拟支付失败"按钮）、过期未提、重复核销、商品下架。
5. 各端均有"重置数据"功能，可随时恢复初始模拟数据。
